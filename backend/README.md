# Attenda Express.js Backend

This is a basic Express.js backend for the Attenda project.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the server:
   ```bash
   npm start
   ```

The server will run on [http://localhost:3001](http://localhost:3001) by default.
